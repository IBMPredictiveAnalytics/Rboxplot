Generate Boxplot Using R Graphics

Boxplots are available in standard IBM® SPSS® Statistics graphics. R boxplots offer some different options. You can classify by one or more categorical variables, add notches, make the box widths proportional to the case count, scale the box widths, and choose box colors.

To create a boxplot:

► Select a scale or ordinal level variable to plot.

► Select one or more categorical variables. All combinations will appear in the plot.

Optionally, you can:

• Add notches to each box. If the notches on two boxes do not overlap, it is very likely that the median values for the two categories are different.

• Make the box widths proportional to the number of cases in the category.

• Make the boxes run horizontally. By default they are vertical.

• Make the boxes wider or narrower relative to the size of the chart by specifying a scale factor. Values smaller than 1 shrink the boxes.

• Specify a title for the chart. Note that the axes will be labelled by the names of the variables.

• Specify the background colors for the boxes. If there are fewer colors entered than needed, they will be reused. Color names must be enclosed in quotation marks and separated by a comma--for example, "red", "green", "blue". You can use any color names recognized by R. Most common names will work.
